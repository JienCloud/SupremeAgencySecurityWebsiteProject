Django File Structure

manage.py

Main command file used to run the server, create apps, and manage the project.

mysite

The main project folder created by Django.

setting.py

Stores project settings like installed apps, database, and templates.

urls.py

Controls which URL goes to which view.

personals and templates

- Your app folder where you write your own code.

- Where HTML files go if you want to make webpages.

views.py

Contains functions like this one that decide what to show to the user.

models.py

Used to create database tables and data structure.

In short:

URL -> View -> Response
